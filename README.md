# Hovid Skill Assessment
This is a web-based assessment system built using ASP.NET (VB.NET). The project is designed to manage user data with basic CRUD (Create, Read, Update, Delete) functionality.

**Features**
1) Add new user records
2) User listing in GridView
3) Edit existing user records
4) Delete user records
5) SQL Server database
6) Simple and responsive UI

**How to Run**
1) Clone this repository
2) Open solution in Visual Studio
3) Restore database
4) Update connection string in Web.config
5) Run project using IIS Express

**How to Restore Database**
1) Open SQL Server Management Studio (SSMS)
2) Open the script file: /Database/hovid-skill-assessment.sql
3) Run the script in SQL Server
